Here are two easy ways to get your website online so you can share it with Sanjana!

## Option 1: GitHub Pages (Free & Easy)

Since you already have the code on GitHub (assumed, as this is a repository), this is the simplest way.

1.  Go to your repository's **Settings** tab on GitHub.
2.  Click **Pages** in the left sidebar menu.
3.  Under **Source**, select the `main` branch from the dropdown.
4.  Click **Save**.
5.  Wait a few minutes.
6.  Your site will be live at: `https://your-username.github.io/valentine-website-2026` (or whatever your repo name is).

## Option 2: Netlify (Recommended for Custom URLs)

Netlify is great if you want a custom link name like `sanjanas-valentine.netlify.app`.

2.  Click **"Add new site"** â†’ **"Import an existing project"**.

5.  Click **"Deploy site"**.
6.  Once deployed, click **"Domain settings"** to change the random name to something special!
